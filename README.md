# MindZone

MindZone is a digital mental-health journaling web application that helps users track emotions, write personal journals, practice mindfulness exercises, and analyze mood patterns.

## Features
- User login and registration
- Typed journaling
- Handwritten journaling canvas
- Mood tracking with notes
- Breathing exercises
- Analytics dashboard

## Tech Stack
- React
- Axios
- CSS
- Node.js / Express (backend)

## Known Issues
Some features are still under development:

- Community posting is currently a placeholder
- Some analytics charts are not fully implemented.
